# RS Car Renter

Premium self-drive car rental website based in Addalaichenai, Eastern Province, Sri Lanka.

## Tech Stack

- HTML5, CSS3, JavaScript (Vanilla)
- Swiper.js (reviews carousel)
- Google Fonts (Orbitron, Inter)

## Features

- Responsive fleet showcase with 5 vehicles
- Car recommendation quiz
- Customer reviews carousel with Swiper
- WhatsApp booking integration (form + individual car buttons)
- FAQ accordion
- Scroll-reveal animations
- Stats counter animation
- Dark theme design

## SEO

- Open Graph / Twitter Card meta tags
- JSON-LD structured data (LocalBusiness, FAQPage, BreadcrumbList)
- Sitemap (`sitemap.xml`) and robots.txt
- PWA manifest (`site.webmanifest`)
- Favicon set (SVG, ICO, PNG, Apple Touch)

## Structure

```
RS/
├── index.html
├── css/style.css
├── js/main.js
├── img/          (vehicle & hero images)

├── robots.txt
```
